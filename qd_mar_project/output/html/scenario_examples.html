Path Scenario Examples

本页用于人工检查不同清算压力方向与价格路径反应的组合是否被正确分类。每行是 24h 主路径窗口中某个 path_pressure_name × path_context × path_label 组合的代表性样本。

读法：directional context 用 path_transmission_ratio 判断传导、吸收、拒绝；neutral / mixed context 用 path_active_z 与 path_active_dominance_score 判断主动主导。

典型场景散点图

横轴为窗口内 signed effective PLIE 累计，纵轴为路径收益。左上/右下通常是压力拒绝或反向接管；横轴接近 0 但纵轴很大是主动主导。
