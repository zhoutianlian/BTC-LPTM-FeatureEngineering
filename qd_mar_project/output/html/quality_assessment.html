QD-MAR Output Quality Assessment

本页用于判断吸收率是否可作为 Agent 输入。核心判据：directional_core 内 q65 coverage 是否稳定、response percentile OOS 是否接近训练分布、directional 证据是否新鲜、Agent 输入是否存在过强冗余，以及标签与 HMM 状态退出是否符合金融机制。
